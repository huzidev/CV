# Huzi-Dev CV

A web-based CV created using Jade (Pug) and Sass, compiled with Prepros.

## Technologies

- **Jade (Pug)**
- **Sass**
- **Prepros** (compiler)

## CV Includes

- **Contact and Profiles**
- **Skills**
- **Education Details**
- **Experience**
- **Download Button** for downloading the CV in JPEG and PDF formats
